# Usage
Available commands
```bash
python todoist-prioritizer.py -h
```

Example usage
```bash
python todoist-prioritizer.py -a 3x4mpl34p1k3y -h 00 -m 00 -d
```

Reset to default settings
```bash
python todoist-prioritizer.py -r
```
